# TheoQuotesAPI
### An API for quoes by theologians

<br />

`/quotes`

Returns all quotes

<br />

`/authors`

Returns a list of all authors

<br />

`/quotes_from/<author name>`

Returns all quotes from the given author

<br />

`/random`

Returns a random quote
